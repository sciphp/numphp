Requirements
------------

NumPhp supports PHP 7.2, 7.3 7.4 and 8.0.

________________________________________________________________________

Install
-------

```
composer require sciphp/numphp
```
________________________________________________________________________

Basic usage
--------------

```php
use SciPhp\NumPhp as np;

$m = np::ar(
  [[ 1, 0, 0],
   [ 0, 1, 0],
   [ 0, 0, 1]]
)->dot(42);

echo "m.42 =\n$m";

```
will output:

```
m.42 =
[[ 42,  0,  0 ],
 [  0, 42,  0 ],
 [  0,  0, 42 ]]
```
